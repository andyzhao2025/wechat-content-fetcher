能源调度算法MPC及场景的开源库

工策

✅ 1. 微电网（Microgrid）能源调度

1.1 MicrogridMPC（基础微电网 MPC）
GitHub: https://github.com/AoufNihed/MicrogridMPC
说明：针对微电网的线性 MPC 控制示例，调度光伏、储能与负载。

1.2 ol‑ems（在线学习增强型 MPC）
GitHub: https://github.com/vittpi/ol-ems
说明：结合在线负载/电价预测与 MPC，用于微电网能源管理控制。

1.3 microgrid‑dispatch‑simulator（微电网调度仿真）
GitHub: https://github.com/leejt489/microgrid-dispatch-simulator
说明：微电网滚动时域调度仿真平台，可集成 MPC 调度策略。

✅ 2. 建筑 / 社区 能源管理系统（EMS）

2.1 MPC‑based‑Energy‑Management‑System（经典 EMS MPC）
GitHub: https://github.com/Nicolas1325/MPC-based-Energy-Management-System
说明：针对建筑/社区 EMS 的 MPC 能源调度示例代码，含储能、光伏与负载管理。

2.2 REC‑Energy‑Management‑MPC‑RL（智能 EMS：MPC + RL）
GitHub: https://github.com/pierlucaanania/REC-Energy-Management-MPC-RL
说明：融合强化学习与 MPC 的能源管理，实现智能优化调度。

✅ 3. 热能 / 热网络 能源调度

3.1 PyREmatcher（热网络 MPC 负载移位）
GitHub: https://github.com/richplane/PyREmatcher
说明：MPC 控制用于热网络负载移位，可扩展至建筑热能调度。

✅ 4. 通用 MPC 工具箱（可用于任意能源调度问题）

4.1 do‑mpc（通用 Python MPC 工具箱）
GitHub: https://github.com/do-mpc/do-mpc
说明：功能完备的 MPC 框架，支持非线性 MPC、经济 MPC、稳健 MPC，适合开发能源调度模块。

4.2 pyMPC（轻量 Python MPC 库）
GitHub: https://github.com/forgi86/pyMPC
说明：轻量级 MPC 库，可用于快速构建优化控制器（适用于小型能源系统调度）。

✅ 5. 智能 / 学习型 MPC 能源调度

注：这一类多与上面的场景交叉，本栏目补充突出学习增强部分。

5.1 ol‑ems（在线学习 + MPC；微电网）
GitHub: https://github.com/vittpi/ol-ems
说明：结合在线学习、预测模型与 MPC 优化，用于微电网。

5.2 REC‑Energy‑Management‑MPC‑RL（MPC + 强化学习；社区 EMS）
GitHub: https://github.com/pierlucaanania/REC-Energy-Management-MPC-RL
说明：强化学习增强的 MPC，用于提高调度效果与策略自适应能力。
